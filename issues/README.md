**All 19 slices are HITL** (Human-in-the-Loop): each slice gets a QA review before the next starts. You give real-time feedback; I revise; we tag the phase milestone when done.

```mermaid
flowchart LR
    Slice[implement slice N] --> PR[open PR<br/>squash-merge to main]
    PR --> QA[you QA the slice<br/>curl + tests + Ragas if applicable]
    QA -->|approved| Next[move to slice N+1]
    QA -->|changes needed| Slice
    Next --> Tag{end of phase?}
```
